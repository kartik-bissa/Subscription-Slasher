# 🚀 Subscription Slasher – AI Expense Analyzer

An AI-powered web application that helps users detect, analyze, and optimize their subscription expenses.

---

## 📌 Overview

Subscription Slasher allows users to upload their bank transaction data (CSV/JSON) and uses AI to:

- Detect recurring subscriptions
- Analyze spending patterns
- Provide smart recommendations to save money

---

## ✨ Features

- 📂 Upload CSV/JSON transaction files
- 🤖 AI-powered subscription detection
- 📊 Interactive dashboard (monthly & yearly costs)
- 📈 Visual charts (category-wise & cost analysis)
- 💡 Smart insights & recommendations
- 💬 AI chatbot for financial advice (optional)
- 🔐 Secure login & authentication (optional)

---

## 🧠 How It Works

1. User uploads transaction file
2. Backend processes data using Pandas
3. AI (Gemini API) detects recurring subscriptions
4. Dashboard displays:
   - Total monthly spend
   - Yearly cost
   - Subscription breakdown
5. AI generates insights & suggestions
